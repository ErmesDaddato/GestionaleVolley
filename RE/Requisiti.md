# Requisiti Funzionali


### Area Gestione Segreteria

RF1: Creazione Account Atelta
Il sistema dovrà permettere alla Segreteria di crere le credenziali per un nuovo account Atleta 

RF2: Creazione Account Allenatore
Il sistema dovrà permettere alla Segreteria di creare le credenziali per un nuovo account Allenatore

RF3: Modifica Account - informazioni
Il sistema dovrà permettere alla Segreteria di mofificare qualsiasi tipologia di account

RF4: Validazione Certificato medico agonistico
Il sistema dovrà permettere alla Segreteria di accettare un certificato medico caricato dall'Atleta

R5: Modifica Blocco Account Atleta
Il sistema dovrà permettere la modifica del blocco di un account Atleta (blocca se il certificato medico non è valido)
 


### Area Gestione Atelta
RF6: Blocco Account - Pagamenti
Il sistema dovrà automaticamente bloccare un account Atleta nel caso in cui questo non risulti in pari con i pagamenti

RF7: Caricamento certificato medico agonistico
Il sistema dovrà permettere ad ogni Atleta già registrato di caricare il certificato medico agonistico

RF8: Caricamento documenti
Il sistema dovrà permettere il caricamento dei documenti di indentità ad ogni Atleta
